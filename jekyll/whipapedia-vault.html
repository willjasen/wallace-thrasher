How the data moves

Scrape → compare → merge

01

Scrape

Fetch a track page from Talkin' Whipapedia and save its human-curated transcript as an immutable run snapshot.

02

Compare

Align wiki dialogue with local subtitle lines, calculate similarity, and propose speaker or text corrections for review.

03

Merge

Apply approved high-confidence changes to the catalogue after preserving the previous local files in a dated backup run.
